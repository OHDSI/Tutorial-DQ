# Data Quality Tutorial for OHDSI

This repository is a hands-on tutorial project for OHDSI users who want to learn how to use the DataQualityDashboard (DQD) against an OMOP CDM. Please see the dqd-tutorial.qmd for more details.

The powerpoint deck "Mastering Data Quality Dashboard for the OMOP CDM.pptx" will take you through the theory of the DQD and the synthetic data used in the example. It also has detailed examples using the data available in this repo describing how to inspect and remediate a selection of data quality checks.
